LXC RUBYGEM

RubyGem for controlling local or remote Linux Containers (LXC)

EXAMPLES

Given the following code:

require 'lxc'

lxc = LXC.new
lxc.version
c = LXC::Container.new(:lxc => lxc, :name => 'test')
c.running?
c.exists?
